🔐 PassGen Pro — генератор паролей с настраиваемыми правилами
Мощный генератор паролей с гибкой настройкой правил: длина, набор символов, исключение похожих и двусмысленных символов, количество паролей, оценка стойкости.
Реализован на 7 языках программирования для демонстрации алгоритмов генерации и пользовательского интерфейса.

🧠 Концепция
PassGen Pro — это инструмент для создания надёжных паролей с учётом современных требований безопасности:

✅ Настраиваемая длина — от 4 до 128 символов.

✅ Выбор типов символов: буквы (верхний/нижний регистр), цифры, специальные символы.

✅ Исключение похожих символов (например, O и 0, l и 1, I).

✅ Исключение двусмысленных символов (например, {}[]()).

✅ Генерация нескольких паролей за раз (до 100).

✅ Оценка стойкости (энтропия и время взлома).

✅ Копирование в буфер обмена (в консоли — просто вывод).

✅ Сохранение настроек в конфигурационный файл.

✅ Интерактивный режим с меню или аргументы командной строки.

🚀 Как запустить
Каждая версия работает из командной строки и поддерживает аргументы или интерактивный режим. Примеры:

bash
# Сгенерировать пароль длиной 16 символов со всеми типами символов
passgen -l 16 -all

# Сгенерировать 5 паролей, только буквы и цифры, длина 20
passgen -l 20 -letters -digits -count 5

# Интерактивный режим
passgen
Установка и запуск для каждого языка описаны ниже.
